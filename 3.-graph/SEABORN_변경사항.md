## 주요 변경사항

### 1. Import 추가
```python
import seaborn as sns
```

### 2. 전역 스타일 설정 (`setup_fonts` 함수)
- seaborn 스타일: `ticks` (세련된 테두리 스타일)
- 폰트 크기 조정: `talk` 컨텍스트 + 0.85-0.9배 (프레젠테이션 수준)
- 색상 팔레트: `husl` + 커스텀 모던 색상
- **한글 폰트 호환성**: rc 파라미터로 Malgun Gothic 유지

```python
sns.set_style("ticks", rc={'font.family': 'Malgun Gothic', 'axes.unicode_minus': False})
sns.set_context("talk", font_scale=0.85)
```
